Archivo Black font setup

I added the code and Minecraft font JSON for Archivo Black.

Because font files should not be redistributed inside this generated zip, this project does NOT include ArchivoBlack-Regular.ttf.

Before building, copy your own font file here:

1.21.11/src/main/resources/assets/durbinclient/font/archivo_black.ttf

Use the file from your uploaded Archivo_Black.zip:
ArchivoBlack-Regular.ttf

Rename it to:
archivo_black.ttf

Then run GitHub Actions / Gradle build.
